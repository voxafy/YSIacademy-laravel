# Feature Audit

## Источник миграции

Исходный проект для миграции:

- `C:\xampp\htdocs\YSIacademy`

Laravel-версия собрана отдельно:

- `C:\xampp\htdocs\YSIacademy-laravel`

## Аудит функциональных блоков

### 1. Публичная часть

- landing page с hero, описанием сервиса и каталогом программ;
- общий header;
- каталог курсов;
- карточка курса;
- страница урока;
- login / register.

### 2. Навигация и layout

- единая верхняя панель;
- разные наборы пунктов для гостя, админа, руководителя, стажера;
- быстрый доступ к профилю и logout;
- переключение темы на фронтенде.

### 3. Auth flow

- вход по email / password;
- выход из аккаунта;
- регистрация стажера;
- хранение текущей сессии;
- блокировка неактивных пользователей;
- role-based middleware.

### 4. Student flow

- кабинет стажера;
- список назначенных курсов;
- открытие карточки курса;
- последовательное прохождение уроков;
- mark lesson complete;
- прохождение quiz-модулей;
- просмотр результатов и решений руководителя.

### 5. Leader flow

- кабинет руководителя;
- обзор команды;
- карточка сотрудника;
- редактирование сотрудника;
- назначение курсов своей команде;
- создание стажера;
- принятие решения по сотруднику;
- комментарии к решению.

### 6. Admin flow

- обзор администратора;
- управление пользователями;
- создание и редактирование профиля пользователя;
- управление курсами;
- создание курса;
- копирование курса;
- удаление курса;
- управление модулями и уроками;
- редактор урока;
- привязка quiz и медиа;
- банк вопросов;
- назначения;
- результаты;
- медиатека.

### 7. LMS logic

- курсы;
- модули;
- уроки;
- lesson blocks;
- видеоблоки и вложения;
- финальные и модульные quiz;
- попытки;
- подсчет прогресса;
- статусы enrollment / progress;
- leader decisions.

### 8. Медиа

- загрузка файлов;
- загрузка видео;
- хранение в локальном storage;
- метаданные в БД;
- безопасная выдача через маршрут `/media/{assetId}`.

## Статусы и сценарии

Используются и сохранены статусы:

- `NOT_STARTED`
- `IN_PROGRESS`
- `FAILED`
- `COMPLETED`
- `SENT_TO_REVIEW`
- `RECOMMENDED_FOR_ACCESS`
- `REPEAT_TRAINING`

Leader decisions:

- `RECOMMEND_ACCESS`
- `RETRAIN`
- `REPEAT_TRAINING`

## Что проверено в Laravel-версии

- `GET /`, `GET /login`, `GET /courses`
- вход под `ADMIN`, `LEADER`, `STUDENT`
- кабинеты `/admin`, `/leader`, `/student`
- страницы `/admin/courses`, `/admin/users`, `/admin/results`, `/admin/media`
- страницы `/leader/assignments`, `/leader/team/{id}`
- карточка курса и страница урока
- mark lesson complete
- submit quiz
- выдача demo-медиа через `/media/{assetId}`
